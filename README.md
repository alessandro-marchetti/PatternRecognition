This repository contains explanations and code snippets for reproducing various figures from Christopher Bishop's _Pattern Recognition and Machine Learning_ (2006).

The material is primarily intended for students taking courses that cover the topics presented in the book. The motivation for creating this repository comes from a homework assignment in one of my graduate courses, which involved implementing the Variational Bayes approach described in Chapter 10.

Not all figures from the book are included. In particular, some figures are omitted because the corresponding datasets were unavailable, while others - especially those illustrating probability distributions outside Chapter 2 - were considered less relevant for the goals of this project. As I complete the remaining chapters, the corresponding code and explanations will be added to the repository. In chapter 10, the derivation for the Variational Gaussian Mixture Model is only partially presented since there were more computations than I expected, so if one wishes to know better, just write to me, or I know there are many other online sources or sites that can guide you.

If you find any mistakes, inaccuracies, or opportunities for improvement, please feel free to get in touch or open an issue. While correctness and clarity were the primary goals, code efficiency was not always the main focus.
